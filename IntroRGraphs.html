ggplot package

ggplot package is one of the most popular packages in R. It offers significant graph options, and its logic is very easy. Consider you are playing with LEGO bricks.

Each ggplot graph has 7 elements:

  1. Data
  2. Graph type (Geoms)
  3. Stats
  4. Scales
  5. Faceting
  6. Theme
  7. Coordinates

Graph Types

  • Bar: geom_bar()
  • Boxplot: geom_boxplot()
  • Histogram: geom_histogram()
  • Line: geom_line()
  • Point: geom_point()…
library(ggplot2)
data(mtcars)
ggplot(data = mtcars, aes(x = mpg, y = hp)) + geom_point()

Graph types - Adding legend

ggpubr packlage has some fucntions to improve graphs.

library(ggpubr)
data(mtcars)
par(mfrow = c(1,2))
g1 <- ggplot(mtcars) +
  geom_point(aes(x = mpg, y = hp, colour = gear)) 
g2 <- ggplot(mtcars) +
  geom_point(aes(x = mpg, y = hp, colour = as.factor(gear)))
ggarrange(g1, g2, ncol = 2, nrow = 1)

Graph types - Managing legend

data(mtcars)
g2 <- ggplot(mtcars) +
  geom_point(aes(x = mpg, y = hp, colour = as.factor(gear)))

g2 + guides(colour=guide_legend(title = "Gear",
                                title.position = "top", keywidth = 2, nrow = 3),
                        expand_limits(x = c(0,35), y = c(0,350))) +
  labs(title="Graph", x = "Miles per gallon", y = "Horse Power")

Graph Types - Faceting and theme

g2 + guides(colour=guide_legend(title = "Gear",
                                title.position = "top", keywidth = 2, nrow = 3),
                        expand_limits(x = c(0,35), y = c(0,350))) +
  labs(title="Graph", x = "Miles per gallon", y = "Horse Power") +
  facet_grid(. ~ carb) + theme_bw()
